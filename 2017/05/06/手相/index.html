手相

发表于 2017-05-06 | 分类于 随笔 | | 次

他是个俗人，却又追求看手相这种玄乎的东西。

他欣赏一句话，“得抑郁症的人都是心地善良的人”，因为他自己抑郁了整整两年。空虚的两年之内，他想过自杀，但思维缓慢营养不良的他因为嗜睡症懒得出去，又因为觉得跳楼割脉太血腥放弃这没有美感的回归自然怀抱的方式。

嗜睡的时候，他认为自己在沉思。他偶尔会梦到关于生命的意义和归宿的主题。梦醒后，去网络上查阅各种各样的奇葩言论。甚至有一天，他会认为所有的精神病患者的一些想法是正确的，或许有些艺术家能佐证这个观点。他可不想成为艺术家，因为他觉得这样会成为一个疯子。

他接触到了手相，因为他某次回家，无聊的时候在家里翻到了一本书。

自从模模糊糊，懵懵懂懂，迷迷茫茫的获取了一些手相知识后，他貌似不抑郁了，因为开启了他招摇撞骗的“神”涯。

“你你你，我来帮你看看手相呗。” 他抖动着他那灵活的眉毛，眯着眼睛，谄谄的望着碰到的同事。

“啧啧，不错不错。生命力旺盛！能活大岁数！但是… 咳咳，不说了。”

“啊哦吆，你这情人不少啊，挺花心啊。看你面相不像啊。”

“嗯嗯，人纹入兑宫，您老啊，老牛吃嫩草啊。”

或许这些类似调侃的话，有人能信，他能忽悠一顿饭来。但大部分人都一笑置之。

他也给自己看。他有时会怔怔望着自己密密麻麻的手纹，心想“这是多么细腻、多么多愁善感的人才能生出这个艺术品”。他幻想自己的手纹能变成油画、甚至是标本。

自从他出关已经1年，今天他跑完步回来洗了个澡。他伸直了手掌，隐隐约约的少了很多纹。他觉得他的抑郁症好了。

然后，他闭上眼，合上了手掌。
