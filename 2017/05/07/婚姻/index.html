婚姻

发表于 2017-05-07 | 分类于 随笔

他是个洒脱的人。他也有自己内心的羁绊，也有自己不舍的依恋，但那是很久之前。

他曾经相信爱情，觉得那就像黑暗中突然亮起的一簇火，很耀眼很明亮，能把自己燃烧。直到后来，他意识到，爱情不是火，自己要变成火，去燃烧给她带来温暖。他有时候觉得这种付出很愉悦，毕竟充实有意义。管最终是不是能感动别人，对得起自己不后悔就行。偶尔他也把这种经历当作以后创造灵感的源泉。

他有时候会远远地静静地站在路灯下，让灯光拉长自己的身影，让路过的人认为他是一个有故事的人。而他，确实有些故事。他会考虑过往，多么怀旧。他希望能找个路灯一样的人，能陪他静静的站着，就这么一直站着。他站久了，也会望望天空的月亮，星星或者偶尔经过的飞机。越深的夜，它们越亮，但亮不过身边的路灯啊。

他不喜欢理发，除非是临近考试或者准备去见亲朋好友。因为长头发才能证明他是自然卷。而他喜欢听理发师说，啊，自然卷。貌似自然卷的人要聪明很多一样。但是他习惯平平常常的毛寸头，因为干净利索。

爱情是什么东西，婚姻是什么东西？他怕已经深入思考会跳跃到 人生是什么东西这种可畏的论题上。

他对婚姻有自己的看法吧。他想到时候再说，再次自诩自己为一个洒脱的人。
